# 关键词匹配策略

## 算法

加权关键词匹配：题目文本 + 选项文本 vs 知识点关键词列表。

- 完整关键词匹配（`\bkeyword\b`）：+3 分
- 部分匹配（keyword 作为子串出现）：+1 分

每题分配给得分最高的知识点。得分为 0 的题目归入第一个知识点。

## 关键词设计原则

1. **覆盖对应题目的高频词**：知识点相关的题目里最常出现的词汇
2. **区分相近知识点**：如"产品创新"和"工艺创新"的关键词应有区分
3. **不要太泛**：不要用"创业""创新"这种通用词
4. **不要太窄**：确保能匹配到足够多的题目（目标 >= 3 题/知识点）

## 举例

知识点"Timmons创业过程模型"的关键词：
```
["Timmons", "创业过程模型", "机会", "团队", "资源", "三要素", "动态平衡"]
```

题目标题："Timmons创业过程模型的核心要素是（）"
- "Timmons"：完整匹配，+3
- "创业过程模型"：完整匹配，+3
- "三要素"：不匹配（题目用的是"核心要素"）
- 总分：6

另一个知识点"商业模式九要素"对此题得分 0。

## 常见问题

**Q: 某个知识点匹配了太多题（>100）怎么办？**
A: 考虑拆分知识点。比如"商业模式"可以拆成"商业模式概念""商业模式九要素""商业模式类型"。

**Q: 某个知识点匹配了太少题（<3）怎么办？**
A: 扩展关键词范围，或考虑与相近知识点合并。

**Q: 大量题目匹配到第一个知识点怎么办？**
A: 第一个知识点（通常是"课程概述"类）的关键词太泛了。收紧关键词，或增加更多区分度的词。
